课程简介

我们本阶段是WEB的手工测试项目阶段。测试项目是基于一个开源的电商平台系统tpshop进行。在本阶段中，我们将介绍如何从入门开始，对一个实际的项目进行比较全面的测试。

学习目标

  • 掌握项目基本测试流程
  • 根据项目需求完成功能点的测试用例设计编写
  • 了解非功能测试
  • 掌握Fiddler工具的使用
