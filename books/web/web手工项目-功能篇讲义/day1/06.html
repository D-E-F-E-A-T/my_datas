测试准备：数据库与项目的关联

思考项目和数据库有什么关联？

举例：注册功能

注册的用户信息是存在数据库里的

连接数据库

  • 打开Navicat 连接到tpshop的数据库tpshop2.0

  • 打开tpbshop后台工具模块，有对应的项目数据库表的列表信息

熟悉项目中的数据库表

数据库也是项目的重要组成部分，在进行项目测试前要先熟悉下项目中重要的数据库表。

在本项目中后面课程中涉及的数据表：
+    首页导航栏内容对应的数据库表为tp_navigation表
+    商城会员对应的数据库表为tp_users表
+    商品分类对应的数据库表为tp_goods_category表 
+    商城商品对应的数据库表为tp_goods表  
+    用户购物车对应的数据库表为tp_cart表

在公司实际项目中可通过开发人员直接获取相关的数据库详细设计信息文档。

  • Tpshop项目中的数据表信息详见《TPshop表结构》
