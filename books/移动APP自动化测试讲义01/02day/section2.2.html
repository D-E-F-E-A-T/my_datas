一、 Appium客户端安装

1.1 Appium背景介绍

```
    1. 官网：www.appium.io,由SauceLab公司开发；
    2. Appium是一个开源、跨平台的测试框架，可以用来测试原生及混合的移动端应用；
    3. Appium是由nodejs的express框架写的Http Server；
    4. Appium使用WebDriver的json wire协议，来驱动Apple系统的UIAutomation库、
       Android系统的UIAutomator框架；

```

  • 2.2.2 Appium-python库安装

    说明：Appium-Python-Client库封装了有关手机操作的API；

    • 命令行安装(需要联网) 【推荐】

        pip3 install Appium-Python-Client

    • 安装包安装 【了解】

        前提：python已安装setuptools包
  安装setuptools:
      1.解压setuptools-38.2.4.zip
      2.进入解压后文件夹执行命令: python setup.py install
      3.等待安装完成，无错误信息即可

  安装Appium-Python-Client:
      1.解压Appium-Python-Client-0.25.tar.gz
      2.进入解压后文件夹执行命令: python setup.py install
      3.等待安装完成，无错误信息即可
