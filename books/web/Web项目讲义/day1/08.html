课程简介

首页测试

页面测试要考虑界面上的显示以及功能点的正确性，除此之外，网页中部分数据内容加载自数据库，界面测试其中之一是验证界面显示数据是否与数据库中一致，有没有达到预期的显示效果。

首页商品分类

根据需求文档《TPshop需求说明书.docx》要求测试首页商品分类功能。

商品分类对应的数据库表为tp_goods_category，根据需求编写对应sql查询语句如下：

所有商品分类
select * from `tp_goods_category` where `is_show` = 1 order by `sort_order`;

  • 商品分类一级分类

    select * from `tp_goods_category` where `is_show` = 1 and `level` = 1 order by `sort_order`;

  • 商品分类的二级分类(部分举例)

1、查看家用电器下的二级分类
select * from `tp_goods_category` where  `is_show`=1  and `level` = 2 and parent_id =2  order by `sort_order`;

2、查看服装下的二级分类
select * from `tp_goods_category` where  `is_show`=1  and `level` = 2 and parent_id =5  order by `sort_order`;

3、查看化妆品下面的二级分类
select * from `tp_goods_category` where  `is_show`=1  and `level` = 2 and parent_id =6  order by `sort_order`;
  • 得到查询结果与页面显示比较查看是否一致
  • 商品分类的三级分类，以化妆品下对应的口腔护理为例
查看化妆品下对应的口腔护理下的三级分类
select * from `tp_goods_category` where `is_show`=1 and `parent_id`=43  order by `sort_order`;

首页轮播图

根据需求文档《TPshop需求说明书.docx》要求测试首页轮播图功能。

首页商品楼层

商品楼层对应的数据库表为tp_goods_category，顶级分类设置为推荐时才会在楼层中显示，推荐字段为tp_goods_category表中的is_hot字段。

取消所有楼层，将所有顶级分类的is_hot字段设置为0，对应SQL如下：

update `tp_goods_category` set `is_hot` = 0  where `is_show` = 1 and `level` = 1

将家用电器和电脑两个顶级分类在楼层中显示，对应SQL如下：

update `tp_goods_category` set `is_hot` = 1 where  `parent_id_path` = '0_2' or `parent_id_path` = '0_3'

商品楼层数据及楼层热门推荐对应的数据库表为tp_goods_category和tp_goods。

家用电器楼层数据，从两表结合筛选出的数据中取中间部分值，对应SQL如下：

select a.*
from tp_goods as a left join tp_goods_category as b on a.cat_id=b.id 
where a.is_hot=1 and a.is_on_sale=1 
and b.parent_id_path like '0_2%'
limit 0, 6;

家用电器楼层热门推荐，滚动展示，每栏显示4个。两表结合筛选出热门推荐商品数据，对应SQL如下：

select a.*
from tp_goods as a left join tp_goods_category as b on a.cat_id=b.id 
where a.is_recommend=1 and a.is_on_sale=1 
and b.parent_id_path like '0_2%'
